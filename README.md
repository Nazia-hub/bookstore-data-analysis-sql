# Bookstore SQL Analysis Project

This project analyzes a bookstore database using three datasets: Books, Customers, and Orders.  
Using SQL, I explored sales trends, customer behavior, and inventory insights.

## Datasets
- **Books**: Book details (title, genre, price, stock)
- **Customers**: Customer information (name, city, country)
- **Orders**: Purchase transactions (order date, quantity, total amount)

## Key Insights
- Best‑selling books and genres
- Top revenue‑generating books
- Customer purchase patterns
- Genre‑wise and author‑wise performance
- Inventory levels and restocking needs

## SQL Skills Used
- Joins across multiple tables
- Aggregations (SUM, COUNT, AVG)
- Grouping and filtering
- Subqueries and ranking
- Date‑based analysis

This project helped me practice real‑world SQL analysis and understand how relational data connects to business insights.
